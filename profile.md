# Minimal Peil
— ミニマリズムとパーマカルチャーで暮らしを整える、いっぺいの記録 —

このブログは、「ミニマルに、自分らしく、土から離れずに暮らす」  
そんな日々の実践や気づきを、そっと記録していく場所です。

ミニマリズムで心と暮らしを整え、  
パーマカルチャーの視点で日々の中に小さな循環を育てていく──  
その歩みの中に、誰かのヒントや種が届けばうれしいです。

自分のペースで、日々の気づきや工夫を記録しています。

- 🧠 ASD/ADHDあり。ひとつのことをじっくり深掘りするのが得意
